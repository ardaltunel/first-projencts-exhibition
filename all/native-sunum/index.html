Native Hakkında Herşey

Web sitelerinin ve web uygulamalarının aksine, native mobil uygulamalar bir tarayıcıda çalışmaz, kendi yerel işletim sistemlerinde çalışacak şekilde geliştirilir.

Popüler Uygulamalar Genelde Native'dir

Kaynakları Belirli Bir Platforma Ayrılır

Twitter, Waze, WhatsApp, Facebook, Instagram, Skype, UberEats, Pokemon Go

Native Uygulamanın Avantajları

Dezavantajları Nelerdir?

Avantajları: Hız, İşlevsellik ve Kalitedir Dezavantajları: Maliyet ve Uzmanlık

Native Uygulamalar Nasıl Geliştirilir?

Native uygulamalar geliştirmek için kendi teknolojilerini kullanır. Android için Java, Kotlin ve C++ iOS için Objective-C veya Swift

Native Web Uygulaması

Native uygulamaların içinde web sitesi açılabilmektedir. Örnek: Kargo şirketlerinin kargo takip için verdikleri sayfa çoğu zaman mobil uyumlu aslında birer web sayfasıdır.

React Native

React Native için Facebook tarafından üretilen, cross-platform mobil uygulama geliştirme olanağı sağlayan bir framework diyebiliriz.
